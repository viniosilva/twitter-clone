# TSLego Nest Hello

## Description

Typescript default Nest with CRUD user. :)
Only copy and paste this folder and start coding!

## Installation

```bash
make install
```

## Running the app

```bash
# locally
make dev

# docker
make up
```

- http://localhost:3000/api/docs

## Test

```bash
# unit tests
make test

# test coverage
make test/cov
```
